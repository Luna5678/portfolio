Latest Projects

Outlet

Freeform art application built for the Mintbean Hackathon in November 2021.

Prompt: Build an app that lets people express their creativity in a visual format. You can build ANY application you'd like, as long as the user can drag my mouse across the screen to make free-form lines.

Result: Fully responsive front-end application. Integrated ability to draw freehand, lines, rectangles, circles in any color. Added eraser, download, and a clear button with a confirmation modal for good user experience.

Cravings

As an avid Yelp user, I've always had trouble finding the latest restaurant menu. Cravings solves that. You can search popular Los Angeles, CA restaurants by cuisine, dietary restrictions, or name. When one interests you, the current menu with current prices are promised so any surprises left are for your taste buds only.

Fully responsive full stack application. Create your own profile, write reviews, and check out where other food enthusiasts have gone to satisfy their cravings.
